Activity 1: Let's Make a Game!

Challenge: Make your own version of Pong

Create a single-player game where you can move a paddle to keep a bouncing ball from hitting the floor.
This is based off of the classic arcade game, Pong.

See your handout for instructions and ask Kyle if you have any questions! This is a fun project!
